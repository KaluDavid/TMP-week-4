HTML TAGS DESCRIPTION
Tag Inline/Block Uses
figure Block Used for representing images, illustrations, diagrams, etc., with optional captions.
div Block A generic container for grouping elements and applying styles.
hr Block Represents a horizontal rule, used for visual separation.
table Block Used for creating tabular data.
form Block Used for creating input forms for user interaction.
fieldset Block Groups related form elements together.
address Block Represents contact information for the author or owner of a document.
iframe Block Embeds an external resource or web page within the current document.
img Inline Embeds an image in the document.
input Inline Used for various types of form input fields.
button Inline Represents a clickable button.
label Inline Labels a form element.
select Inline Creates a dropdown list.
strong Inline Represents strong importance, typically displayed as bold text.
a Inline Creates a hyperlink to another page or resource.
span Inline A generic inline container used for grouping and styling inline elements.
